This Opus package contains models for integrating the Emme/2 travel model and UrbanSim.

Requirements:

  • Define environment variable TRAVELMODELROOT to contain the absolute path to the directory containing your emme/2 travel model bank directory, e.g. "C:" if you have a directory "C:/baseline_travel_model/2000_c05/tripgen. Note that you can use either forward or backward slashes in the directory path.
