PSRC Indicators

This folder contains a set of python scripts for creating indicator results for the PSRC 2000 baseyear.  See the python scripts for the actual database being used.

To run these indicators, you will need to setup your computer with the appropriate supporting software:

  1. Python.
  2. Python Indicator Libraries.

You can choose to run these indicators either:

  • Directly from the python command prompt - follow these instructions. 
  • Or from within the Eclipse development environment - follow these instructions.

To choose which indicators to compute:

  1. Open indicators_to_run.py.
  2. Comment out the ones you do NOT want to run.
  3. Run indicators_to_run.py via python.

Note: This functionality will also be available in the indicator browser, when that is up and running.
