The Data Menu is mainly used for processing and importing the data required for a particular project into MySQL database tables and for viewing and modifying them. It has the following options available in it.

Data Source Connection
This option can be used to changed the MySQL database connection settings for a particular project.

Import
The next step after the project setup involves the importing of data into the MySQL database for further processing and running the synthesizer. This can be done using the Data -> Import menu option or by selecting the green down arrow Import icon in the toolbar. Importing the data involves the following 6 steps:

  1. Processing Geographic Correspondence data
  2. Processing Housing PUMS/ Sample data
  3. Processing Person PUMS/ Sample data
  4. Processing Housing Summary data
  5. Processing Person Summary data
  6. Processing Region's Shape file
The import process begins by clicking on Start. The completion of each step is indicated by a check symbol beside it. The first 5 steps download the census files if required, parses them and then uploads them into the MySQL database as tables. During this process, you will be prompted to overwrite any files if they already exist and also for the selection of the specific variables to be imported into the database. The final step downloads and extracts the shape files containing the boundaries of the geographies selected.

View and Modify
This option is activated only when a particular table in the Data Tables section of the File Manager area is selected. It pulls up the data table to be viewed and edited.
