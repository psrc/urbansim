This Opus package exists to operate on the set of Opus packages that we distribute together.  This set of Opus packages is defined in two places:

  • the <copy-this-project> in the fireman.xml
  • the opus_package_names list in tests/all_tests.py

To change the set of Opus packages to put into the Opus base distribution, change both of these lists.

This package does the following:

  • Runs the tests/all_tests.py file for each included Opus package.
  • Creates a zip file containing all of the included Opus packages (without any .pyc files)
