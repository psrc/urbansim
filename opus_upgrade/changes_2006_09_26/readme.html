To create the buidling_types table and the buildings table, do the following steps in this sequence:

  1. Flatten your scenario database chain. Execute opus_core.database_management.flatten_scenario_database_chain. In addition to other parameters, you will need to specify your scenario database and the name of the new database that represents the flattened chain. Important: throughout the rest of the scripts that you run, you will use this new database name as the database.
  2. Download the scripts from changes_2006_09_26_code.zip and unzip into a temporary directory on your computer.
  3. Execute the do_create_building_types_table_for_elcm.py script to create a building_types table; you'll have to specify the database name, and perhaps other parameters (see below for usage syntax), e.g.:
python do_create_building_types_table_for_elcm.py -d my_database_name
  1. Execute the do_create_buildings_table.py script to create the buildings table, e.g.
python do_create_buildings_table.py -d my_database_name

To see usage information, execute 'python script_name --help', e.g.:

python do_create_building_types_table_for_elcm.py --help
