20 September 2006

To adjust a database for the modifications to the ELCM, do the following steps in this sequence:

  1. Flatten your scenario database chain. Execute opus_core.database_management.flatten_scenario_database_chain. In addition to other parameters, you will need to specify your scenario database and the name of the new database that represents the flattened chain. Important: throughout the rest of the scripts that you run, you will use this new database name as the database.
  2. Download the scripts from elcm_changes_2006_09_20_code.zip and unzip into a temporary directory on your computer.

  3. Execute the do_rename_by_swapping_employment_and_commercial_or_industrial_for_elcm.py script.  This script:
    • Renames employment_commercial_location_choice_model_specification to
      • commercial_employment_location_choice_model_specification.
    • Renames employment_commercial_location_choice_model_coefficients to
      • commercial_employment_location_choice_model_coefficients.
    • Renames employment_industrial_location_choice_model_specification to
      • industrial_employment_location_choice_model_specification.
    • Renames employment_industrial_location_choice_model_coefficients to
      • industrial_employment_location_choice_model_coefficients.

  4. Execute the do_copy_home_based_to_single_and_multiple_family_home_for_elcm.py script.  This script:
    • Splits employment_home_based_location_choice_model_coefficients into
      • single_family_home_elcm_coefficients and
      • multiple_family_home_elcm_coefficients.
    • Splits employment_home_based_location_choice_model_specification into
      • single_family_home_elcm_specification and
      • multiple_family_home_elcm_specification.

  5. Execute the do_create_building_types_table_for_elcm.py script.  This script:
    • Creates a building_types table

  6. Execute the do_combine_job_for_estimation_tables_for_elcm.py script.  This script:
    • Creates jobs_for_estimation from
      • jobs_for_estimation_commercial and
      • jobs_for_estimation_industrial and
      • jobs_for_estimation_governmental and
      • jobs_for_estimation_home_based.

To see usage information for any of these scripts, execute 'python script_name --help', e.g.:

python do_create_building_types_table_for_elcm.py --help
