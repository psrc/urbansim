11 April 2007

To convert an UrbanSim cache, or any directory containing "flt_storage" files, produced by the numarray version of UrbanSim (version 4.0 or before) into one that can be read by the numpy version of UrbanSim (version 4.1 or after), do the following steps in this sequence: 

Note: This must be done on a computer that has the same "endian-ness" as the computer that created the cache. (It is highly unlikely that you need to worry about endian-ness unless the computer that created the cache was using an PowerPC, Motorola or SPARC processor. For more info, see Wikipedia.)

  1. Download the scripts from changes_2007_04_11_code.zip and unzip into a temporary directory on your computer.
  2. Run the do_convert_numarray_cache_to_numpy_cache.py. The first parameter is the path to the directory to copy (the destination). The second is the path into which the copy will be put (the output directory). This utility will not modify the original directory or its files.

For example, if you are trying to convert a cache directory on Windows at C:\cache\old_run to a new one located at C:\cache\converted_run, you would execute:

python do_convert_numarray_cache_to_numpy_cache.py -d "C:\\cache\\old_run" -o "C:\\cache\\converted_run"

In the case of a problem, or if your environment variables have not been set, see usage information by executing:

python do_convert_numarray_cache_to_numpy_cache.py --help
