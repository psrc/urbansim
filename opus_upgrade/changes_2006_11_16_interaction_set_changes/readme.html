To convert the interaction set names, do the following steps in this sequence:

  1. Download the scripts from changes_2006_11_16_interaction_set_changes_code.zip and unzip into a temporary directory on your computer.
  2. The set of changes that will be made are listed in the substitutions list in the substitutions.txt file.  If you have additional interaction datasets that are not in that list, add additional lines before you run the script.
  3. Execute the do_interaction_set_changes.py script to update the names of the interaction sets in the variable_name field of all specification tables in the given database, e.g.:
python do_interaction_set_changes.py= -d my_database_name -t my_table_name

The -d and -t options may be used multiple times within the same command to specify additional databases or tables.

In the case of a problem, or if your environment variables have not been set, see usage information by executing:

python do_interaction_set_changes.py --help
