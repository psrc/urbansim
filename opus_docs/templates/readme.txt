This zip file contains the Python source code for the principal packages
for Opus and Urbansim for revision {revision}.  Please see
docs/installation/index.html for installation instructions.  The docs/
directory also contains tutorials and the Opus/UrbanSim user guide for this
revision.

The source code for all the Opus/UrbanSim packages can be found in our
subversion repository at https://trondheim.cs.washington.edu/svn/opus --
you can check out either the latest revision, or a prior one.  (If you
check out the revision corresponding to this zip file you can get the code
in the packages in this zip file in the form of a subversion repository
checkout, as well as additional packages.)

Please see the project website at http://www.urbansim.org for further
information.
