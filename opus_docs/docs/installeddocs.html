Finding Your Installed Opus / UrbanSim Documentation

If you have installed Opus, this document describes how to access the version of the Opus / UrbanSim documentation that corresponds to the version of the system you are using.  Otherwise, use one of the versions linked from here.

The documentation installed by the Opus installer resides in the opus_core Python package within Python's site-packages directory.  To access these append the following to the location of your Python site-package's directory (e.g. to "/usr/lib/python2.4/site-packages/"):

  • Snapshot (14 July 2006)
    • pdf version - opus/manual/docs/latex/userguide.pdf
    • html version - opus/manual/docs/latex/userguide/index.html
  • Nightly release
    • pdf version - opus_docs/docs/latex/userguide.pdf
    • html version - opus_docs/docs/latex/userguide/index.html
On Windows, if you have installed Python 2.4 to the C: drive, you can use the following links:
  • Snapshot (14 July 2006)
    • pdf version - C:\Python24\Lib\site-packages\opus\manual\docs\latex\userguide.pdf
    • html version - C:\Python24\Lib\site-packages\opus\manual\docs\latex\userguide\index.html
  • Nightly release
    • pdf version - C:\Python24\Lib\site-packages\opus_manual\docs\latex\userguide.pdf
    • html version - C:\Python24\Lib\site-packages\opus_manual\docs\latex\userguide\index.html
