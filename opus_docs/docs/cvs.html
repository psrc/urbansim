Getting Opus Packages from CUSPA’s CVS Repository

The following directions assume that you are using Eclipse to connect to CVS and that you have an account on the University of Washington Computer Science & Engineering computers. (At the moment, this is the only way to connect to our CVS repositories, though we plan to fix that.)

  1. In Eclipse, open the CVS Repository Exploring perspective by clicking onWindow!Open Perspective!CVS Repository Exploring
  2. Right-click on the CVS Repositories window and select New > Repository Location
  3. Enter the following information:
    • Host: trondheim.cs.washington.edu
    • Repository path: /projects/urbansim2/repository
    • User: <your CSE UNIX account name>
    • Connection type: extssh
  4. Press Finish
  5. You will be prompted for password for your CSE UNIX account. Enter it and press OK.
  6. In the CVS Repositories view, expand the repository whose name ends with ":projects/urbansim2/repository", open the repository’s HEAD, select the projects you want to check out by right-clicking on each of them, and selecting Check Out. This set typically includes: opus_core and urbansim.
