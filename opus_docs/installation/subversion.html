Getting Opus Packages from CUSPA's Subversion Repository

The following directions assume that you are using Eclipse to connect to subversion.

  1. In Eclipse, open the SVN Repository Exploring perspective by clicking on Window!Open Perspective!SVN Repository Exploring
  2. Right-click on the SVN Repositories window and select New > Repository Location
  3. Enter the following information:
    • URL: https://trondheim.cs.washington.edu/svn/opus/
  4. Press Finish
  5. In the SVN Repositories view, expand the repository location, select the projects you want to check out by right-clicking on each of them, and selecting Check Out. See Opus Revisions and Releases for an explanation of trunk, tags, etc. The projects you want typically includes: opus_core, urbansim, eugene, and opus_gui (and perhaps others).
