Choosing a Development Environment

In general, if you are doing much software development, you will benefit a lot from using an integrated development environment (IDE) that supports for the range of activities involved in, such as such as editing code, running tests, debugging code, connecting to a code repository, etc.  There are numerous such IDEs for Python, and no one IDE fits all needs.  The software developers in CUSPA typically use Eclipse for talking to subversion, Wing for debugging, and either or both for editing Python code.
