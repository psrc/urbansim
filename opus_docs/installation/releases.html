Opus Revisions and Releases

Periodically we make a stable release of Opus and UrbanSim. These stable releases have version numbers such as 4.2.0 (where this is for Opus/UrbanSim version 4, minor number 2, bugfix number 1). In addition, we have a development version of the system, which we update very frequently. Most users will want the latest stable release of the system; developers and others who are working closely with the core development team will want the current development version. All of these releases are available from our subversion repository at https://trondheim.cs.washington.edu/svn/opus/.

The stable releases are under this URL: https://trondheim.cs.washington.edu/svn/opus/tags/. For example, version 4.2.0 is at https://trondheim.cs.washington.edu/svn/opus/tags/4.2.0/.

The current development version is under this URL: https://trondheim.cs.washington.edu/svn/opus/trunk/.

Finally, you will notice a branches directory under https://trondheim.cs.washington.edu/svn/opus/branches. This is for branches off of the main trunk -- generally users shouldn't need to be concerned with those. (Branches are used, for example, for stable releases -- a given release is just a tagged revision in a branch.)
